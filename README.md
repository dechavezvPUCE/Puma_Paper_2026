# Puma_Paper_2026
This repository contains all the necessary scripts to replicate the results presented in the paper by Chavez et al. (2026), published in Scientific Reports.  For more information, please contact [danielchavez@uti.edu.ec](mailto:danielchavez@uti.edu.ec).  
First, run the submit script using either sbatch or qsub, depending on your supercomputer's job scheduling system.
